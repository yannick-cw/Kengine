# kengine

A search engine in Haskell, built as a learning project alongside reading [Designing Data-Intensive Applications (2nd edition)](https://www.oreilly.com/library/view/designing-data-intensive-applications/9781098119058/) by Martin Kleppmann.

An HTTP server that indexes documents against a strict mapping and serves ranked, fuzzy, and semantic search queries. Each phase of the [curriculum](curriculum.md) maps to DDIA chapters and delivers a usable increment, powering local product search for [korb](https://github.com/yannick-cw/korb), my REWE grocery pickup CLI.

This project was set up using the approach described in [Jokes on you, AI: LLMs for Learning](https://www.dev-log.me/jokes_on_you_ai_llms_for_learning/).
